Gallery folder setup:

Each gallery section now has its own folder.

Use these folders:
- images/gallery/pikes-peak/
- images/gallery/chca-dirt-racing/
- images/gallery/pavement-racing/
- images/gallery/youtube-videos/
- images/gallery/team-presence/
- images/gallery/partner-exposure/

How to add photos:
- Put the main tile image for that section in the folder as cover.jpg
- Put up to 6 section photos in the same folder as 1.jpg, 2.jpg, 3.jpg, 4.jpg, 5.jpg, and 6.jpg

Example:
- images/gallery/pikes-peak/cover.jpg
- images/gallery/pikes-peak/1.jpg
- images/gallery/pikes-peak/2.jpg

Notes:
- This is the simplest folder-based setup for GitHub Pages.
- Because the site is static, it still needs a simple naming pattern inside each folder.
- If a photo is missing, the page will still show a styled placeholder block.
