# O'Maley Racing Website

Static multi-page website ready for GitHub Pages.

## Live site files

- `index.html`
- `about.html`
- `team.html`
- `partners.html`
- `results.html`
- `journal.html`
- `gallery.html`
- `styles.css`
- `images/`

## Images

Put all website images in the `images` folder.

See `images/README.txt` for the exact file names used by the site.

## Publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload all files in this folder to the repository root.
3. In GitHub, open `Settings > Pages`.
4. Under `Build and deployment`, choose:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main` and `/root`
5. Save and wait for GitHub Pages to publish the site.

## Custom domain

When you are ready to connect your domain:

1. Add your domain in `Settings > Pages > Custom domain`.
2. Create the DNS records your domain provider gives you.
3. After the domain is confirmed, GitHub can manage HTTPS for the site.

## Notes

- This site does not require WordPress.
- Missing images will fall back to styled placeholders until you add the real files.
- If you want, the next step can be adding your final branding, copy, and real photos.
